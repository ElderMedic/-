# SBhandover
A descriptive document for new department principle

## ※ 生保简介

华中农业大学信息学院学生会生活保障与权益部致力于服务我院学生，主要负责生活信息发布、维护学生权益、执行内务检查、安全卫生与后勤保障、失物招领等工作，部门在校内院内广泛开展维权活动，配合各部门完成各种保障任务。生保部工作范围广泛，发挥独一无二的作用，长期奋战在学生工作第一线，处理疑难问题，与师生、学校耐心交流，传达意见建议、发布学校通知消息，是重要的校园网宣阵地和沟通渠道，不仅是学生平时接触最多的学生工作部门之一，也是深受广大师生欢迎和爱戴的部门。

> 生保部的工作特点是范围广、内容繁杂、重复性强，考验管理人员耐心、细心、恒心，对于为人处世、与人交流沟通的能力也是一种磨练，在众多学生工作职务中属于比较冷门、受排斥的，对于负责人既是机遇也是挑战。*本手册内容以我的任期执行工作的回忆为准。*

## ※ 部长职权

- 领导部门人员完成工作，进行决策，布置任务，维持部门正常运转
- 协调内外关系，联系院内院外领导与隶属的相关人员、组织
- 代表学院参加校部、院会组织的会议，加入相关群聊、组织，传达重要消息通知，解答疑问，传达意见
- 控制部门人力事务，组织招新，监督干事、副部工作
- 参加学代会，参加相关会议
- 拟定部门及隶属人员的考核条例
- 提交部门月、学期考核
- ···

### >日常工作：安全卫生检查
- 院部例查

每周一次，一般是每周五下午，范围大一到大三寝室
需要提前开条，模板打印好找辅导员签字，然后去综合楼1楼宿管中心盖章，凭条到各栋宿管处拿钥匙，可能需要压学生证在宿管处待查完归还
还需要提前准备打分表

一般情况下，一栋楼仅需一名到两名干事进行检查，取钥匙后上楼逐户敲门、表明身份、进入检查，按照校部发布的安全卫生检查标准打分，我们的打分偏好一般如下：

情况|描述|判分
---|:--:|---:
优秀|没异味，垃圾桶无过多垃圾，地上桌上均干净整洁，无违规用电，叠被子床边未挂过多衣物|95+
良好|标准中一项有不合格，整体观感良好|90+
一般|2项以上，多人不合格情况，整体观感差|80-89
差|整体感受差，完全不敢进门，多项多人不合格|70+
一般不会打70以下，若有违规用电须加重扣分，没收违规电器，我们的评分与其他院相比总体偏低

没收的违规电器可自行保存，或放到逸夫楼，通知相应年级辅导员，过一段时间可归还

> 打分表须周一前做好电子版发到nice生保大家庭群文件相应文件夹中（注意打分表需要有平均分），并邮件发送所有辅导员+孙超书记+中心主任，也可qq发送。纸质版打分表集中保存。
> 校部并不要求检查大三寝室卫生，只需要查安全情况，也不必每周上交大三检查分数，但是我们一直坚持都查。

- 院部抽查

若辅导员没有过问可少执行，抽查即抽取一部分寝室突击检查，时间寝室需要与检查人员、女生部、辅导员（一般辅导员会想去）协调，提前确定但保密，第一次进行可以在生委群预告。其余事项与例查相同。

- 学校抽查

由学工处、各院辅导员、楼管会组织进行，对生保部保密，一般由楼管会和其他院辅导员一起进行，仅通报检查处罚结果，结果一般发在华农宿管群，一定要关注转发到生委和寝室长群，有问题可向宿管群老师反映。一般查到违规电器会直接没收，然后执行限电措施，三次内是一个月限电400w。

- 违规电器

一般我们检查第一次发现不没收，怕得罪同学们，口头提醒收起来，但是决定权在你。
校部没收的一个月后会通知辅导员让人来荟六3楼楼管会办公室去取，需要登记，一般没收的锅、壶、卷发棒不少，建议2个人去。可以自己存放也可以放在荟五3楼半活动室、逸夫楼，需要告知辅导员。


### >日常工作：晚归检查、人数清查
晚归的检查主要是每周日晚上进行，我们需要用短信和表格报送全院所有人的晚归情况，是否到校到寝室。短信需要在21:30前发给潘锐老师，遵循以下格式：

> *信息学院应到861人，实到846人，未到14人。15级应到188人，实到181人。16级应到188人，实到185人。17级应到211人，实到209人。18级应到274人，实到272人。未到校原因主要是交通原因在家（路上），交流，实习或者病假等。新疆西藏籍少数民族学生应到0人，实到0人，未到0人。*

表格需要做两种，一个word版一个excel版，信息本质上是一样的，我现在让干事做word班我做excel版。周一晚发给辅导员qq和邮箱，还是每个辅导员、主任、孙书记一人一份。

检查过程已经简化了，我们只需要派人检查荟十大四的人（以后是荟七荟八），大一到大三交给了网格员，但是他们报的不是很准时，需要催。让在网格员群的干事注意报送情况，同时部长要注意女生部发过来的女生大四的晚归信息，及时转发给负责干事。

### >日常工作：生委、寝室长

平时的通知消息都要转发到生委群、寝室长群，重要的让生委转发班群通知各班，同时回答生委和寝室长的问题。也可以安排给副部干事做。
生委需要组织班级的一些活动，比如收学生证、火车票优惠卡充磁等，尽量给他们一些事情做。
每年换届时期需要及时了解各班生委人选，召开至少一次生委例会（我只开了一次），提前制作生委考核条例在会上公示、发放、讨论，收集生委联系方式，需要给新生讲解生委和寝室长只能。寝室长人太多暂不需要考核开会，但是需要每年统计各个寝室长信息，也比较麻烦。

### >工作周志

简要介绍部门上周工作、本周开展的工作等，重点是待办重大事项，我们一般没什么好写的，有模板，每周开部长例会前一晚需要发给秘书处负责人，相关情况会在部长群通知。

### >火车票优惠卡

比较麻烦的事情，由校部通知和统计、执勤队负责，具体是每学期进行一次充磁、改区间、办理和补办（办理和补办单独进行，和改区间充磁时间上不在一起）

充磁一次有四个磁条（使用机会），按规定每个同学每年也只能用四次，只能往返自己家所在地和武汉的车站（转车须购买联程票），所以一般一年充磁一次即可，但是往往会有人消磁等等。充磁我们不经手，直接交给各班生委负责，他们收齐班级所有人学生证之后去综合楼统一办理，我们只需要通知。

改区间一般是去综合楼的窗口进行，需要改的个人自己进行，他们要先打印一个申请找辅导员签字学院盖章再到综合楼相应窗口办理，我们只通知不需要经手。

火车票的事情麻烦在于大多数人不清楚相关政策和操作方式，我也不知道，但是你可以问校部、执勤队人员，以及查看生委群里已经有的火车票优惠卡QA问题解答的文件（各个群有不少），很多人懒得看还是会问，尽量耐心解答，重点要提醒所有人。

补办和办理是针对每年入学的新生和丢失学生证的老生进行的，大概一学期一次补办，办理也就大一入学的时候有，比较麻烦。需要整理一个大一到大四所有需要办优惠卡的人的信息表，电子版和纸质版都要交过去，而且信息要100%正确无误遵循格式，尤其是乘车区间填写规则（只要站名，不要带其他乱七八糟地名称谓，例如武汉站、汉口站、武昌站）。这个表会改两稿，第二稿只需要电子版，第一稿正确无误才行，校部会指出错误，如果错了全部重来再交一遍纸质版，所以一定要仔细。

> 需要注意的就是火车优惠卡的补办，因为每年会车票优惠卡的补办一般都在学生证补办之前，而且每学期只有一次补办的机会，所以这个一定要提前跟班支他们说好。优惠卡补办的信息统计一般需要统计大一新生的，还有需要补办的老生的信息，所以不仅要特别跟新的大一的班支说补办的事情，也需要特别跟大二、大三、大四的学长学姐们说，每个学期补办的次数只有两次，一般在4、5月份或者11、12月份。

学生证的火车票优惠卡页面需要盖华中农业大学红色圆章，这个章应该是开学学班收取新生学生证（上面已经填好了区间和信息）后去行政楼盖的，但是如果没盖需要赶紧跟辅导员说然后他交一个申请再一起去盖，这个比较严肃，我去年给两个班的新生盖了很久，所以你最好加学班群跟学班讲清楚，他们可能会有不负责的。

武汉人不享受火车票优惠卡，也不要填写学生证上的乘车区间（最好空着），但是如果是你去盖可以给本地人优惠卡那一面盖章，其实没人管你是不是武汉人，只要家庭地址填的是外地都可以改区间享受优惠（相当于骗优惠，这是个漏洞）。

### >假期留校

这一件事情也比较繁琐，包括每年暑假寒假前统计留校的同学，给留校同学安排住处，安排男女生负责人，和留校负责人沟通管理留校晚归情况。然后还需要帮忙贴封条封寝室，这件事也可以让负责人自己搞但是一般他们两个人忙不过来。

统计留校一般是校部和辅导员通知，会给你统计表模板，在生委群通知各班统计好发给你汇总即可，然后电子版发到学工处教管科邮箱，纸质版交到综合楼学工处教管科（交完登记）。

一般暑假留校人数较多，各寝室楼也不封楼，每个寝室有2个留校的就可以住原寝室，只有一个人的需要安排住合住在一起，需要你们统计好找他们一个个沟通安排，最后要把住宿情况交上去。寒假会仅开放少数楼，一般男女各一栋，所有留校学生搬进去住，需要统计好封楼期间留校的同学，报上去然后学校给分配房间，然后我们再安排房间住的人，有可能信息学院的不在一起，一定要每个寝室都有个呆的旧的，一个寝室里的人留校时间相近，不要落单。一般指定留校时间长的当各个寝室负责人，让他们去领取钥匙。

男女生负责人一般由辅导员找，这个是有工资的，还会有留校网格员负责报晚归，晚归一般由负责人搞，生保不需要管。

封楼就是所有没人留校的寝室贴封条，学校会检查，一般贴的时候大家都走了，需要我们协助。封条其实可以自己撕下来进去住，只要没封楼就不会停电停水，具体听通知。

### >日常工作：伙委

包括伙委帮厨和值班，帮厨基本上没有班级搞，可以发动生委带各班有兴趣的联系食堂经理参与一次，可以算一次公益活动，对我们月考核也有加分。
值班就是饭点派人到食堂伙委值班桌坐着，中途不准玩手机吃东西，最好看看书什么的，结束后需要填一个在线表格（伙委群每天中午发），报一下伙委的失物信息，然后带回交到校部。
